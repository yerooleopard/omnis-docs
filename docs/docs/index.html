# General

Welcome to Omnis WordPress Theme Docs. You will find here the all necessary information how to use this Theme. If you have any questions feel free to send a message via this contact form.

# Requirements

To install this Theme you must have working version of WordPress 5.0 or higher. If you need help installing WordPress follow the instructions in official WordPress Codex.

# In Package

List of Items contained in "Main File(s)" package purchased and downloaded via themeforest.net:

  • Omnis Theme Folder – contains omnis.zip" which is main installable zip file of Omnis Theme and "omnis-child.zip" - Omnis Child Theme. Information about child themes you will find here.
  • Docs Folder – contains Omnis Theme, Revolution Slider and WPBakery Visual Composer documentation.
  • Licensing Folder – GPL Licensing informations.
