# Installation

You can install Omnis Theme in three ways - With Envato Market WordPress Plugin, directly with WordPress (easier and faster way) or via FTP.

# Installation with Envato Market Plugin

  1. Install Theme with Envato Market Plugin - go to Envato Market Plugin Page to learn more.
  2. Go to After Installation Process.

# Installation in WordPress

  1. In WordPress Admin Panel go to Apperance -> Themes. Click Add New Button, then click Upload Theme. Installation in Wordpress
  2. Select omnis.zip file and click Install Now Button
  3. Wait until installation process will be done.
  4. Go to After Installation Process.

INFO

Sometimes you might have "max upload filesize" problem. To solve this problem follow instructions in this article.

# Installation via FTP

  1. Unzip omnis.zip file into omnis folder. Make sure that in omnis folder you don't have any other subfolder – only theme files and subfolders.
  2. Log in to your hosting service via FTP Client (e.g. FileZilla)
  3. Find there WordPress installation and then go to wp-content -> themes
  4. Upload omnis folder into themes. Wait until you have uploaded all theme files. Installation via FTP
  5. Go to After Installation Process.

# After Installation

  1. Activate Theme right after installation or go to Appearance -> Themes and click Activate Button Theme Activation
  2. Go to Appearance → Install Plugins then install & activate required plugins.
    Read more about recommended plugins here. Plugins Activation
  3. Go to Appearance → Menus. Create new menu and assign it to Header - Main Navigation location. Optional: Create another menu and assign it to Header - Secondary Navigation.
    Read more about menu options here Menu Assign
